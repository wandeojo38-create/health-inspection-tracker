# Health Inspection Tracker

A comprehensive smart contract system for managing public health inspections of food facilities.

## Key Features
- **Facility Registration**: Register restaurants, cafes, and food establishments
- **Inspector Authorization**: Authorize health department inspectors
- **Inspection Management**: Conduct inspections with scoring and status tracking
- **Violation Tracking**: Record and track health code violations with severity levels
- **Public Reporting**: Transparent access to facility inspection status
